### Semantics in HTML 5

Semantic HTML or semantic markup is HTML that introduces meaning to the web page rather than just presentation.

|  标签   | 说明  |
|  ----  | ----  |
| aside  | 导航性质的工具内容 |
| article  | 具有明确的独立性的内容 |
| hgroup  | 标题组 |
| abbr  | 当前内容为缩写 |
| hr  | 表示故事走向的转变或者话题的转变 |
| p  | 普通的段落 |
| strong  | 表示重要性 |
| blockquote  | 表示段落级引述内容 |
| q | 表示行内的引述内容 |
| cite  | 表示引述的作品名 |
| time  | 表示时间 |
| figure  | 表示与主文章相关的图像、照片等流内容 |

| dfn  | 包裹被定义的名词 |
